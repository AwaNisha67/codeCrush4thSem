SQL SELECT Statement

The SELECT statement is the most commonly used command in Structured Query Language. It is used to access the records from one or more database tables and views. It also retrieves the selected data that follow the conditions we want.


By using this command, we can also access the particular record from the particular column of the table. The table which stores the record returned by the SELECT statement is called a result-set table.

Syntax:-

SELECT col1, col2,...,colN FROM table_name;

To access all rows from all fields of the table then we use SELECT statement with * asterisk sign:-

SELECT*FROM table_name;

SQL SELECT Statement example:-

Creating table name student from which we will select the rows.

CREATE TABLE student(rollNo INT PRIMARY KEY,
studName VARCHAR(20),
lastName VARCHAR(20),
age INT CHECK(age >= 0 AND age <= 20));

Inserting some Values in the above created table:-

INSERT INTO student
        (4265,"irfaan","khan",20),
        (4266,"adhiraj","kanwar",19),
        (4267,"awanisha","pal",19);

Selecting the values from the student table:-

SELECT* FROM student;
