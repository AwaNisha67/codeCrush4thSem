SQL MIN() and MAX()

The Min() function is used to return the minimum value of the selected columns, and The Max() function is used to return the maximum value of the selected columns.

MIN() Function Syntax :-

SQL min function returns the smallest value in the column. The MIN() function provides the smallest value of the chosen column.

SELECT MIN (column_name)
FROM table_name
WHERE condition;

MAX() Function Syntax :-

SQL max function returns the largest value in the column. The MAX() function provides the largest value of the chosen column.

SELECT MAX (column_name)
FROM table_name
WHERE condition;
