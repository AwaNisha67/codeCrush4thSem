SQL CREATE TABLE

In the SQL database for creating a table, we use a command called CREATE TABLE.

SQL CREATE TABLE Statement

A Table is a combination of rows and columns. For creating a table we have to define the structure of a table by adding names to columns and providing data type and size of data to be stored in columns.

Syntax

CREATE TABLE table_name(
column1 datatype(size),
column2 datatype(size),
column3 datatype(size),
.
.
columnN datatype(size));

SQL CREATE TABLE example

CREATE TABLE student( 

rollNo INT PRIMARY KEY, 

studName VARCHAR(20),

lastName VARCHAR(20),

age INT CHECK(age >= 0 AND age <= 20),

phNum INT 

);

Create Table Using Another Table

To create a copy of an existing table we use can also use CREATE TABLEcommand.In the new table, it gets the exact column definition all columns or specific columns can be selected.

Syntax

CREATE TABLE table_name AS
SELECT column1, column2,..columnN
FROM existing_table_name
WHERE (some_condition);

SQL Query:-

CREATE TABLE oddStudent AS 

SELECT * FROM student
 
WHERE rollNO = 1,3;
