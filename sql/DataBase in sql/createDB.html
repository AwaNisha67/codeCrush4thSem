SQL CREATE DATABASE Statement

In SQL we need to create a new database. Only after creating a database can we perform actions in the database like adding tables and records. A database consists of tables in which we store the data or records.

SQL CREATE DATABASE

The "CREATE DATABASE” statement is used to create a new database in SQL. It is also used in MySQL and other relational database management systems (RDBMS) to create databases.

Important Points:

  • We have to type the database name after the CREATE DATABASE statement.
  • The database name is case-insensitive, so we need to create a unique database name.
  • Blank spaces are not allowed in the name of the database, we can only use underscore (_).

Syntax

CREATE DATABASE database_name;

SQL Query:-

CREATE DATABASE codeCrush;

Output:-
